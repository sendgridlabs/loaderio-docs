GET /v2/tests/:test_id/results/:result_id

Get a specific result from this test.

curl https://api.loader.io/v2/tests/f133e9e3691e405eefd3e1b7c351cb18/results/08627345e5aa0f89f95851a69a4cb0dc \
     -H 'loaderio-auth: API_KEY'
